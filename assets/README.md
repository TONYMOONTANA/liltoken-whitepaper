# Assets Folder

This folder contains visual assets, infographics, and branding materials for $LILTOKEN.

## Contents

- **Brand Guidelines**: Logo usage, color schemes (neon green #00ff88, dark blue #1b263b)
- **Infographics**: Token distribution charts, roadmap visualizations
- **Marketing Materials**: Social media templates, banner images
- **Scarface-Themed Visuals**: Blimp imagery, quote graphics

## Usage Guidelines

- All assets are for official $LILTOKEN promotional use
- Maintain brand consistency when creating derivative works
- Community members are welcome to use these assets for fan projects

## Contributing Assets

If you'd like to contribute visual assets:
1. Ensure they align with the Scarface theme
2. Follow the brand color scheme
3. Submit via pull request with clear descriptions
4. Include source files when possible
